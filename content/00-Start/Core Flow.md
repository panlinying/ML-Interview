# Core Flow

This product is practice-first. The core flow has three tracks that run every week.

## Track 1: Coding sets (primary)
- 4-5 sessions per week
- Work through **[[10-Sets/Overview]]** in order

## Track 2: ML Core (refresh + explain)
- 2 sessions per week
- Start at **[[20-ML-Core/Module 01 - Math Foundations + Classical ML]]** and continue in order

## Track 3: System design drills
- 1 session per week
- Start at **[[30-System-Design/Framework & Systems]]**, then case studies

## Session template
1. Warm up (1 old problem)
2. Main set (60-90 minutes)
3. Review (pattern + pitfall + complexity)
