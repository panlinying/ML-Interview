# Practice Home

This is a practice-first interview prep system for experienced ML engineers.

## Start in 10 minutes
1. Read **[[Practice Loop]]**.
2. Pick your first set from **[[10-Sets/Overview]]**.
3. Follow the weekly cadence in **[[Core Flow]]**.

## Core flow (repeat weekly)
- 4-5 coding sets
- 2 ML Core modules
- 1 system design drill

3. Write the pattern + pitfall
